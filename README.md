# opencode-skill-zettelkasten-builder

从零搭建 Obsidian 卡片盒笔记库。面向各行业新手，不预设知识背景。

## 介绍

这是一个 [opencode](https://opencode.ai) skill，帮助你从零开始搭建一套属于自己的 Obsidian Zettelkasten（卡片盒笔记法）系统。不是只搭个架子就走人——在搭的过程中，会教会你这套东西到底怎么用、为什么好用。

## 安装

1. 将 `ZettelkastenBuilder/` 复制到 opencode skills 目录：

   ```bash
   cp -r ZettelkastenBuilder ~/.config/opencode/skills/
   ```

2. 在 opencode 中触发即可使用。

## 使用

| 触发词 | 行为 |
|--------|------|
| "搭建知识库"、"卡片盒入门"、"帮我建个卡片系统"、"从零开始记笔记" | 执行完整 7 阶段流程 |

## 流程

| 阶段 | 内容 |
|------|------|
| Phase 1 | 摸底：了解用户背景、记录习惯、关注领域 |
| Phase 2 | 讲理念：三大原则 + 必要难度理论 |
| Phase 3 | 搭目录建配置：创建 vault 结构、写 Obsidian 配置 |
| Phase 3.5 | 旧笔记自动归类：导入已有素材并按角色分类 |
| Phase 4 | 学语法写模板：Markdown 基础、卡片模板、命名规范 |
| Phase 5 | 亲手制卡：用"两问"筛选，AI 框架引导，用户自己写正文 |
| Phase 6 | 收尾总结：上手清单、生成架构文档、进阶方向推荐 |
