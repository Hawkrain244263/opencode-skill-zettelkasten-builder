# 你的卡片盒使用指南

## 目录结构

```
000_inbox/        — 收件箱：闪念、截图、素材先丢这
100_Calendar/     — 随记、日记、时间线
200_Card/         — 永久卡片（核心知识资产）
300_Extra/
├── 301_附件/     — 图片附件
└── 302_模版/     — 笔记模板
400_Source/       — 外部输入存档（读书笔记、课程笔记等）
500_Space/        — 项目空间（不参与卡片体系）
```

## 卡片命名格式

```
YYYYMMDD_HHmm_中文标题.md
```

例：`20250609_1430_为什么周一总比周五觉得累.md`

短名（引用用）：`1430_为什么周一总比周五觉得累`

## 制卡前的两问

1. 我为什么要写这个想法？
2. 我能拿它干什么？

答不上来就暂时不记。

## 符号说明

```
←  来源/引用  — "我的这个观点来自哪"
→  延伸/去向  — "这个观点可以延伸出什么"
```

## 完整工作流

```
采集（Capture） → 整理（Develop） → 关联（Connect） → 输出（Output）
   000_inbox        200_Card        卡片之间链接      文章/项目
                    400_Source
```
