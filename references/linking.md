# 双向链接规则（供 AI 教学中引用用）

## 符号说明

```
←  来源/引用  — "我的这个观点来自哪"
→  延伸/去向  — "这个观点可以延伸出什么"
```

## 双向链接原则

如果你在卡片 A 写了 → [[卡片 B]]（A 延伸出 B），那卡片 B 里应该有一条 ← [[卡片 A]]（B 引用 A）。

如果不这样做，就会出现"单向链接"——A 提到了 B，但 B 没有回链指向 A。时间一长，这个链接方向就被遗忘了。

## 方向推导

| A 的写法 | B 应补什么 |
|----------|-----------|
| A 有 `← [[B]]`（A 引用 B） | B 应补 `→ [[A]]`（B 延伸至 A） |
| A 有 `→ [[B]]`（A 延伸至 B） | B 应补 `← [[A]]`（B 引用 A） |

## 什么情况不需要双向链接

- **平行关系/类比**：两张卡只是相似，没有先后或因果方向 → 不需要互链，当参照就行
- **同一来源的平行卡片**：比如同一本书的好几张笔记卡片 → 保持独立，不互链
