# 命名规范（供 AI 教学中引用用）

## 卡片文件名

```
YYYYMMDD_HHmm_中文标题.md
```

例子：`20250609_1430_为什么周一总比周五觉得累？.md`

- `YYYYMMDD` = 日期（如 20250609）
- `HHmm` = 时间（如 1430），小写 mm 表示分钟
- `_中文标题` = 下划线后接中文

## 为什么用时间戳不用数字编号

数字编号（1.1、2.3 这种）会让知识按照数字自动归类——1 开头全是设计，2 开头全是心理——这就违背了"不预设分类"的扁平化原则。

时间戳让每张卡都平等，没有谁比谁高级，知识才能自由跨界链接。

## 短名规则（引用时用）

```
HHmm_中文标题
```

例子：`1430_为什么周一总比周五觉得累？`

- 若多张卡片共享同一 HHMM → 用完整 `YYYYMMDD_HHMM_中文标题`
- 短名只在卡片之间的 `[[链接]]` 中使用

## AI 辅助产出命名

```
{SkillName}_{YYYYMMDD}_{HHMM}_{主题关键词}.md
```

例子：`ZettelkastenBuilder_20250609_1430_入门总结.md`
