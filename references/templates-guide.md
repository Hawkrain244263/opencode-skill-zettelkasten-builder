# 模板说明（供 AI 教学中引用用）

## 卡片模板

```markdown

---

关联资源：
← [[]]    — 来源（从哪来的）
→ [[]]    — 延伸（可以连到哪张卡）

延伸思考：
```

各部分含义：

| 部分 | 作用 |
|------|------|
| 标题 | 用疑问句做标题，让人想点进去看。"什么是……？""为什么……？" |
| 关联资源 ← | 来源/引用——这个观点从哪来的（书、文章、课程等） |
| 关联资源 → | 延伸/去向——这个观点可以引出什么（其他卡片） |
| 延伸思考 | 埋问题钩子，让卡片"播种问题"而不是只记录答案 |

## HQ&A 模板

```markdown

---

H
> 

Q

A

```

什么时候用：**读完一篇文章或一本书时**。

| 部分 | 作用 |
|------|------|
| H（Highlight） | 摘录原文关键句 |
| Q（Question） | 给自己提一个问题 |
| A（Answer） | 用自己的话回答 |

重点在 QA 环节——给自己提问再回答。这一步就是"费点劲"的地方，也是跟抄书式笔记最大的区别。
